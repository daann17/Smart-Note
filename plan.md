# SmartNote - 智能个人知识管理与协作平台 需求规格说明书

## 1. 项目概述

### 1.1 项目名称
SmartNote - 智能个人知识管理与协作平台

### 1.2 项目简介
SmartNote 是一个现代化的、智能驱动的个人与企业级知识管理平台。它超越了传统的笔记应用，通过集成全文搜索、人工智能辅助和知识图谱可视化，帮助用户高效地组织、检索、连接和升华他们的知识与想法，并支持安全的团队协作。

### 1.3 技术栈
- 前端：Vue 3、Pinia、Vue Router、Ant Design Vue、Vditor
- 后端：Spring Boot、Spring Security、JWT
- 数据库：PostgreSQL（主数据存储、全文检索、向量存储）
- 缓存：Caffeine（本地缓存）
- AI 能力：集成 OpenAI GPT 或同等级别大模型 API（后期再做）
- 部署：Docker（可选）

## 2. 角色定义
### 普通用户（User）
- 所有个人笔记、笔记本、标签的 CRUD
- 使用 AI 功能
- 查看知识图谱
- 搜索全局内容（仅限自己有权限的）
- 分享笔记和协作

### 系统管理员（Admin）
- 用户管理（增删改查、禁用/启用）
- 系统监控（查看系统负载、存储使用情况）
- 全局数据管理（查看全平台笔记/用户统计）
- AI 服务配置与管理

## 4. 非功能性需求

### 性能
- 页面首屏加载时间小于 3 秒
- 笔记列表/搜索接口响应时间小于 500 ms
- 支持至少 100 个并发用户同时进行笔记编辑操作

### 安全性
- 所有密码加密存储
- 使用 JWT 进行无状态认证，API 接口需权限校验
- 严格数据隔离，用户仅可访问其有权限的数据
- 对用户输入进行验证与过滤，防止 SQL 注入与 XSS 攻击

### 可靠性
- 系统核心功能（笔记编辑、保存）可用性大于 99.9%
- 完善的数据备份与恢复机制

### 可用性
- 界面设计简洁直观，符合现代 Web 应用审美与操作习惯
- 提供清晰的操作反馈与引导

### 可扩展性
- 采用微服务或清晰的模块化架构，便于未来功能扩展
