# SmartNote 项目进度报告

最后更新时间：2026-03-25

## 1. 当前阶段判断

SmartNote 已经从“功能原型”进入“可持续迭代的可用版本”。当前代码库具备完整的前后端主链路，核心体验已经不再依赖单点演示，而是形成了以下几个稳定模块：

- 认证与权限链路已经闭环
- 笔记、笔记本、标签、回收站、版本历史已经可持续使用
- 编辑器已经从 Vditor 切换为 `CodeMirror 6 + Yjs + STOMP` 的 CRDT 协同方案
- 搜索、知识图谱、分享协作、个人中心已经具备前台可见的产品形态
- 管理端开始从“统计碎片”演进到“可操作的管理员中心”

## 2. 模块进度总览

状态说明：

- `已完成`：核心功能已经落地并可正常使用
- `进行中`：已有稳定基础，但仍有明显扩展空间
- `未开始`：代码库中暂未形成对应实现

| 模块 | 当前状态 | 说明 |
| --- | --- | --- |
| 用户认证与授权（AU） | 已完成 | 已实现注册、登录、JWT、路由守卫、接口鉴权、协作者登录门槛、管理员接口权限控制 |
| 个人知识库核心（NK） | 已完成 | 已实现笔记本/笔记 CRUD、Markdown 编辑、标签、回收站、版本历史、移动复制、文件上传 |
| AI 辅助（AI） | 进行中 | 已有摘要、标签建议、AI 助手入口；更深入的知识图谱自动构建与 RAG 仍待继续推进 |
| 搜索与发现（SD） | 进行中 | 已实现关键词搜索、高级筛选、知识图谱页面；搜索高亮、筛选记忆等体验项仍待补齐 |
| 分享与协作（SC） | 进行中 | 已实现公开分享、提取码、评论、CRDT 实时协同、协作者登录要求；段落评论与导出仍未完成 |
| 系统管理与监控（SM） | 进行中 | 已实现管理员概览、用户列表、账号启停、角色调整；存储统计与索引维护仍待补齐 |

## 3. 已落地能力

### 3.1 认证与用户体系

- 用户注册、登录、退出已完成
- JWT 认证与前端路由守卫已稳定接入
- 个人中心已支持维护：
  - 昵称
  - 简介
  - 绑定手机号
  - 生日
- 协作者参与实时协同前必须先登录
- 管理员接口已启用服务端角色鉴权

### 3.2 笔记与知识管理

- 笔记本创建、编辑、删除、移动
- 笔记创建、编辑、保存、预览
- Markdown 编辑与图片/文件上传
- 标签维护与笔记标签关联
- 版本历史、预览与回滚
- 回收站恢复、清空与定时清理
- 最近编辑列表与工作台总览

### 3.3 协同编辑

- 主编辑器已切换为 `CodeMirror 6 + Markdown + Yjs + STOMP`
- 文本同步基于 CRDT，不再使用脆弱的纯文本广播
- 已实现多人同时编辑时的冲突自动收敛
- 已实现协作者在线状态、身份颜色区分与编辑器内中文化
- WebSocket 协同链路与保存接口都带权限校验

### 3.4 搜索与知识图谱

- 首页全局搜索可按关键词查询
- 已支持高级筛选：
  - 按笔记本
  - 按标签
  - 按更新时间起止日期
- 已完成知识图谱独立页面：
  - 笔记本 / 笔记 / 标签节点可视化
  - 节点拖拽与回弹
  - 选中节点固定 / 取消固定
  - 关系显隐切换
  - 右侧详情面板
  - 关联关系按类型分组显示
  - 关系类型快速筛选

### 3.5 分享与协作

- 支持生成公开分享链接
- 支持提取码访问
- 支持分享评论
- 支持登录后加入协同会话
- 匿名用户不再具备协同编辑能力

### 3.6 管理员中心

- 已新增独立管理员页面 `/admin`
- 已实现管理员路由守卫与服务端接口鉴权
- 已实现系统概览卡片：
  - 总用户数
  - 启用账号数
  - 停用账号数
  - 管理员数量
  - 笔记 / 笔记本 / 标签总量
- 已实现用户管理表格：
  - 用户信息查看
  - 角色查看与切换
  - 账号启用 / 停用
  - 自身账号保护
  - 最后一个启用管理员保护

## 4. 与计划书相比的重要变化

- 编辑器技术栈已经从 `Vditor` 切换为 `CodeMirror 6 + Yjs`
- 协同编辑方案已经升级为 CRDT，同步可靠性显著高于传统文本广播
- 知识图谱已不再是占位概念，而是具备交互和详情面板的真实页面
- 管理端能力不再只停留在统计接口，已经形成独立管理员中心
- 个人中心已完成基础资料维护，不应再视为待开发项

## 5. 下一阶段建议优先级

结合当前完成度，下一批最值得继续投入的方向如下：

1. 分享协作深化
   - 段落级评论
   - 协同区域提示
   - 导出 PDF / Word
2. 搜索体验增强
   - 结果高亮
   - 最近筛选条件记忆
   - 更清晰的结果分组
3. AI 能力继续补齐
   - 图谱关系自动建议
   - 更稳的知识问答链路
4. 管理端扩展
   - 存储占用统计
   - 搜索索引维护
   - 更完整的系统监控
